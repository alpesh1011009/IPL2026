## How to Convert to PNG for Instagram

**Option 1 — Browser:**
1. Open SVG in Chrome/Edge
2. Right-click → "Save as image" or take a screenshot at full size

**Option 2 — Inkscape (free):**
```
inkscape post-1-hero.svg --export-png=post-1-hero.png --export-width=1080
```

**Option 3 — Online:**
Use any SVG-to-PNG converter (e.g., svgtopng.com)
